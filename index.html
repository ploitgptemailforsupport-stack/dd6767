Groka UI

v5 Documentation

Window
Creates the main Groka UI window with dragging, minimizing, closing, tabs, and animations.
Syntax
local Window = Library:CreateWindow("Groka Hub", "Best UI Library")
Tab
Creates a tab inside the window.
Syntax
local MainTab = Window:CreateTab("Main", "🏠")
Section
Creates a section divider.
Syntax
MainTab:AddSection("Combat")
Button
Creates a clickable button with ripple effects.
Syntax
MainTab:AddButton("Kill All", "Executes attack", function() print("Clicked") end)
Toggle
Creates a smooth animated toggle switch.
Syntax
MainTab:AddToggle("Auto Farm", "Automatically farms", false, function(v) print(v) end)
Slider
Creates a draggable value slider.
Syntax
MainTab:AddSlider("WalkSpeed", 16, 100, 16, function(v) game.Players.LocalPlayer.Character.Humanoid.WalkSpeed = v end)
Input
Creates a textbox input field.
Syntax
MainTab:AddInput("Player Name", "Enter username", function(text) print(text) end)
Dropdown
Creates an expandable dropdown menu.
Syntax
MainTab:AddDropdown("Select Weapon", {"Sword","Gun","Knife"}, function(v) print(v) end)
Keybind
Creates a customizable keybind.
Syntax
MainTab:AddKeybind("Toggle UI", Enum.KeyCode.RightShift, function() print("Pressed") end)
Label
Creates an editable text label.
Syntax
local Label = MainTab:AddLabel("Status: Idle") Label:Set("Status: Running")
Paragraph
Creates a paragraph card with title and body text.
Syntax
MainTab:AddParagraph("Information", "Groka UI is smooth.")
Notify
Creates animated notifications.
Syntax
Library:Notify("Success", "Loaded successfully", 5, "success")
Loadstring
Loads the Groka UI Library directly from GitHub using loadstring.
Syntax
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ploitgptemailforsupport-stack/GrokaUI/refs/heads/main/MyUI.lua"))()
Example
local Library = loadstring(game:HttpGet("https://raw.githubusercontent.com/ploitgptemailforsupport-stack/GrokaUI/refs/heads/main/MyUI.lua"))() local Window = Library:CreateWindow("Groka Hub", "Best UI") local Main = Window:CreateTab("Main", "🏠") Main:AddButton("Print Hello", "Example button", function() print("Hello") end)
